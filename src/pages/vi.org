#+title: VI

VI, which is not six.

* 操作
- 0 :: 行頭へ
- $ :: 行末へ
- dw :: 単語削除
- d$ :: 行末まで削除
- yy :: yank (copy)
- p :: paste カーソル後に
- P :: paste カーソル前に
- H,M,L :: 画面の上・真ん中・下へ移動
